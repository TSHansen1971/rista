# RISTA-001 – Independent Development Note

Rísta shall be developed as an independent application with its own source code, visual identity, documentation, interaction decisions and implementation choices.

## General product patterns

The following are general and widely used software patterns:

- Markdown as an open plain text writing method
- document-based macOS editing
- side-by-side editor and preview
- keyboard-driven writing
- local file ownership
- ordinary open document formats
- native macOS application behavior

These patterns are treated as general product and platform patterns, not as proprietary identity.

## Third-party material boundary

The project shall not include third-party proprietary material unless explicit rights have been obtained.

This includes:

- source code
- app icons or graphic assets
- App Store text
- screenshots
- proprietary names
- proprietary visual identity
- pixel-near layout, color or interaction details
- distinctive interaction sequences that are not generic platform behavior

## Product identity

Rísta is an Apple Silicon-native macOS Markdown editor for plain text writing, live preview and open `.md` files.

Its identity is based on the name Rísta, the ᚼ mark, native macOS behavior and a focused writing workflow.

## Documentation footer

Repository-facing Markdown documentation shall carry the standard Rísta footer with author notice, repository URL, CC BY-NC-ND 4.0 notice, initial publication date and last modified date.

This footer marks documentation and written pages. It does not by itself finalize the software license for source code.
